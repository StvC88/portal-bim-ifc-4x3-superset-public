# Portal BIM IFC 4x3 + Superset Embebido

Repositorio público espejo del portal BIM local con IfcOpenShell, visor GLB y Apache Superset embebido.

- Portal web: React + Vite + TypeScript
- API: FastAPI
- Extracción IFC: IfcOpenShell + IfcConvert
- Datos analíticos: PostgreSQL
- Dashboards: Apache Superset
